# Secure Notes App - INFT425

A secure note-taking application with biometric authentication and end-to-end encryption.

## Features Implemented

- ✅ Biometric authentication (fingerprint/Face ID)
- ✅ PIN fallback authentication
- ✅ AES-256 encryption for all notes
- ✅ Search notes by title/content
- ✅ Pin notes to top
- ✅ Auto-lock after inactivity
- ✅ Performance optimizations (ListView.builder, const constructors)
